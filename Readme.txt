To change the test image, place a new image name on line 85.
While the program is running, press j to execute the step 1
RBG enhancement, press k to run the step 2 YCrCb enhancement,
and press p to enter selection mode for step 3 ROI enhancement.
Select the ROI and it will be automatically enhanced. Press c
to clear the current selection.
All files will be saved to the current directory.
When finished, press esc to close the windows and terminate
the program.
